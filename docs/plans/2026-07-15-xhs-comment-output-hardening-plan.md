# 小红书评论区输出加固实施计划

- 在 `tests/plugin-main-ai.test.js` 添加评论规范化、结构合并、格式、最终统计和转写末尾规则。

- 规范时间、点赞并按最终 Markdown 重新统计。
- 运行针对性测试直至通过。

## 任务 3：采集循环修复

- 更新诊断语义，网络监听成功时不再把无签名直连失败显示为主故障。
