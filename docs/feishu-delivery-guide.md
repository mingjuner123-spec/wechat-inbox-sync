## 这篇文档解决什么问题

这是「Obsidian 收集箱」的用户交付文档。用户打开小程序里的“插件下载与绑定教程”后，可以在这里完成三件事：

1. 下载并安装 Obsidian 同步插件。
2. 把微信小程序和自己的本地 Obsidian vault 绑定起来。
3. 按需配置语音转文字 API。

当前 MVP 的核心目标很简单：把微信里的文字、链接、语音、网页和文件，稳定收进 Obsidian。

## 一、当前支持哪些内容

| 收集类型 | 小程序入口 | 保存结果 |
| --- | --- | --- |
| 文字 | 读取剪切板 / 手动粘贴 | 原文保存为 Markdown 笔记 |
| 链接 | 读取剪切板 / 手动粘贴 | 链接保存为 Markdown 笔记 |
| 语音 / 音频 | 录音 / 音频 | 保存音频；配置 API 后同时保存转写文本 |
| 网页链接 | 上传 | 抓取网页正文并转为 Markdown |
| 公众号文章 | 上传 | 抓取正文并转为 Markdown |
| 飞书公开文档 | 上传 | 抓取正文、标题层级和可识别图片并转为 Markdown |
| PDF / Word / TXT / MD | 上传文件 | 文件内容转为 Markdown；原文件作为附件保存 |

说明：飞书、公众号、PDF 等内容会受到原网页权限、图片防盗链、文件编码和文档结构影响。如果遇到转换不完整，可以先确认链接是否公开可访问，或把文件直接上传。

## 二、安装 Obsidian 插件

1. 电脑先安装 Obsidian，下载地址：https://obsidian.md/
2. 打开你的本地知识库文件夹。
3. 找到 `.obsidian/plugins/` 目录。如果没有 `plugins` 文件夹，就手动新建一个。
4. 在本文档顶部下载最新版插件。
5. 解压后，把 `wechat-obsidian-inbox` 文件夹放入 `.obsidian/plugins/`。
6. 重启 Obsidian，或在 Obsidian 里重新加载插件。
7. 打开「设置 > 第三方插件」。
8. 如果安全模式还开着，先关闭安全模式。
9. 启用「微信 Obsidian 收集箱」插件。

插件目录示例：

```text
你的知识库/.obsidian/plugins/wechat-obsidian-inbox/
```

## 三、绑定小程序和 Obsidian

1. 在微信小程序首页点击「绑定 Obsidian」。
2. 进入绑定页面后，复制 6 位绑定码。
3. 回到 Obsidian，打开插件设置页。
4. 填写同步 API 地址。使用官方小程序时，所有用户都填写官方同步地址；如果是自己部署的小程序和云函数，才填写自己的云函数 HTTP 访问地址。
5. 填写小程序绑定码。
6. 保存目录默认用「临时收集」即可。
7. 点击「立即同步」。
8. 回到 Obsidian 检查是否生成了新笔记。

首次测试建议按这个顺序来：

1. 先测试一段普通文字。
2. 再测试一个网页链接。
3. 最后测试语音、PDF、Word 或飞书链接。

这样更容易判断是哪一步配置出了问题。

## 四、日常怎么用

### 读取剪切板

适合复制文字、普通网页链接、资料链接。

点击「读取剪切板」后，小程序会读取当前剪切板内容。确认内容没问题后点击「保存」，内容会原样进入 Obsidian。

### 录音 / 音频

适合临时想法、会议片段、语音备忘录。

如果没有配置语音转写 API，Obsidian 里会保存音频播放器。

如果配置了语音转写 API，Obsidian 里会同时保存音频和转写全文。

### 上传

适合上传网页链接、公众号文章、飞书公开链接、PDF、Word、TXT、Markdown 文件。

上传时间可能比文字和链接更长。上传期间请等待页面提示完成，不要重复点击。

## 五、语音转文字方案怎么选

MVP 阶段推荐三种选择：

| 方案 | 适合谁 | 优点 | 注意事项 |
| --- | --- | --- | --- |
| 不开启转写 | 只想保存语音原文件的用户 | 零配置、零成本、最稳定 | 只有音频，没有文字 |
| 豆包语音识别 | 大多数普通用户 | 配置相对简单，有免费额度，国内访问稳定 | 需要开通「录音文件识别 2.0」并使用语音服务 API Key |
| 腾讯云 ASR | 已经熟悉腾讯云的用户 | 长音频能力稳定，控制台成熟 | 密钥、权限、资源包配置相对复杂 |
| 阿里云百炼 Qwen-Omni | 已在使用阿里云 / 百炼的用户 | 一个 API Key 可覆盖多模态能力 | 模型、额度和计费规则要看控制台 |

本地 Whisper 方案暂不放进 MVP。它可以降低云端成本，但安装、显卡、系统环境和电脑适配会让普通用户上手变复杂，后续再单独做。

API 获取入口与教程：

- 豆包语音识别：
  - 控制台：https://console.volcengine.com/speech/new/
  - API Key 使用说明：https://www.volcengine.com/docs/6561/1816214
  - 录音文件识别标准版 API 文档：https://www.volcengine.com/docs/6561/1354868?lang=zh
- 腾讯云 ASR：
  - 语音识别控制台：https://console.cloud.tencent.com/asr
  - API 密钥管理：https://console.cloud.tencent.com/cam/capi
  - 官方开通与调试教程：https://cloud.tencent.com/document/product/1093/104735
- 阿里云百炼 Qwen-Omni：
  - API Key 页面：https://bailian.console.aliyun.com/cn-beijing?apiKey=1#/api-key
  - API Key 获取说明：https://help.aliyun.com/zh/model-studio/get-api-key/
  - API-Key 页面说明：https://help.aliyun.com/zh/model-studio/user-guide/apikey

## 六、豆包语音识别配置

1. 打开火山引擎控制台。
2. 进入「豆包语音」。
3. 开通「录音文件识别 2.0」。
4. 进入「API Key 管理」。
5. 复制语音服务的 API Key。
6. 在 Obsidian 插件设置里选择「豆包语音识别」。
7. 粘贴 API Key。
8. 保存设置后，用小程序录一条语音测试。

常见问题：

- 401：API Key 不对，常见原因是填成了火山方舟大模型 Key。
- 403：服务没开通、资源没授权、免费额度没生效或账号余额异常。
- requested resource not granted：确认插件已经是最新版，并确认开通的是「录音文件识别 2.0」。
- 转写为空：先确认录音本身有声音，再重新同步一次。

## 七、腾讯云 ASR 配置

1. 打开腾讯云控制台。
2. 开通「语音识别 ASR」。
3. 领取免费资源包或确认账号有余额。
4. 创建 API 密钥，拿到 SecretId 和 SecretKey。
5. 确认 CAM 权限允许调用 ASR。
6. 在 Obsidian 插件设置里选择「腾讯云 ASR」。
7. 填入 SecretId 和 SecretKey。
8. 地域默认 `ap-shanghai`，普通话模型默认 `16k_zh`。
9. 保存设置后，用小程序录音测试。

常见问题：

- SignatureFailure：密钥复制错误，重新生成并复制 SecretId / SecretKey。
- UnauthorizedOperation：权限不足，检查 CAM 权限和服务是否开通。
- 没有转写结果：等待任务处理完成，或在插件设置里增加等待次数。

## 八、阿里云百炼 Qwen-Omni 配置

1. 打开阿里云百炼控制台。
2. 创建并复制 API Key。
3. 确认账号有免费额度或余额。
4. 在 Obsidian 插件设置里选择「阿里云百炼 Qwen-Omni」。
5. 填入 API Key。
6. 模型保持默认即可。
7. 保存设置后，用小程序录音测试。

常见问题：

- 401：API Key 错误或复制不完整。
- 403：服务未开通、账号权限异常或额度不可用。
- 返回空文本：检查音频是否可访问，录音里是否有声音。

## 九、插件更新说明

当前版本：0.1.4

更新时间：2026 年 5 月 14 日

本次版本重点：

1. 支持文字、链接、语音、网页、公众号、飞书链接、PDF、Word、TXT、Markdown 文件同步到 Obsidian。
2. 支持语音原文件保存，配置 API 后支持语音转文字。
3. 优化飞书公开文档提取，尽量保留正文结构和可识别图片。
4. 优化 PDF 文本提取，减少竖排乱码和页码干扰。
5. 优化上传状态，文件处理完成前会持续显示处理中，减少重复上传。
6. 优化小程序分享入口。

后续如果插件有更新，会在本文档顶部放新版插件。小程序公告栏会显示「某月某日有插件更新，更新文件请在绑定教程下载」。

更新插件的方法：

1. 在本文档顶部下载最新版插件。
2. 解压后覆盖旧的 `wechat-obsidian-inbox` 文件夹。
3. 回到 Obsidian，关闭并重新启用插件。
4. 点击「立即同步」测试。

## 十、安全提醒

不要在公开群聊、截图、文档里暴露以下信息：

- 豆包 API Key
- 腾讯云 SecretId / SecretKey
- 阿里云 API Key
- 云函数 HTTP 访问地址中的敏感参数

如果已经泄露，建议立刻去对应平台删除旧 Key，并重新生成新 Key。
