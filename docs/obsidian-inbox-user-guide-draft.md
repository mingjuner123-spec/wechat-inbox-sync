# Obsidian 收集箱绑定与语音转写教程（飞书初稿）

## 这篇文档解决什么问题

这篇教程用于放在飞书里，作为小程序首页“绑定教程”和“插件更新公告”的统一入口。用户可以在这里完成 Obsidian 插件绑定、选择语音转写方案，并下载后续新版插件。

## 一、绑定 Obsidian 插件

1. 在电脑上打开 Obsidian。
2. 打开当前知识库的插件目录：`.obsidian/plugins/`。
3. 将 `wechat-obsidian-inbox` 插件文件夹放入该目录。
4. 回到 Obsidian，进入“设置 > 第三方插件”。
5. 如果安全模式未关闭，先关闭安全模式。
6. 启用“微信 Obsidian 收集箱”插件。
7. 在微信小程序首页点击“绑定 Obsidian 插件”。
8. 复制小程序生成的绑定码。
9. 回到 Obsidian 插件设置页，填写：
   - 同步 API 地址：云函数 HTTP 访问地址，通常以 `/sync` 结尾。
   - 小程序绑定码：从小程序复制的绑定码。
   - 保存根目录：默认 `临时收集`。
10. 点击“立即同步”，确认文字、链接或语音可以写入 Obsidian。

## 二、语音转写方案区别

| 方案 | 适合谁 | 成本 | 配置难度 | 说明 |
| --- | --- | --- | --- | --- |
| 关闭转写 | 只想保存原始语音的人 | 免费 | 最低 | 只把 MP3 同步到 Obsidian，不生成文字。 |
| 豆包语音识别 | 大多数 MVP 用户 | 有免费额度，用完后按量计费 | 低 | 推荐默认方案。使用录音文件识别 2.0，长语音更合适。 |
| 腾讯云 ASR | 已经在用腾讯云的人 | 有免费额度，用完后按量计费 | 中 | 需要开通 ASR 权限和 CAM 权限，配置项更多。 |
| 阿里云百炼 Qwen-Omni | 已经在用阿里云百炼的人 | 有免费额度，用完后按量计费 | 中 | 走大模型多模态接口，适合后续扩展总结整理。 |

MVP 阶段建议：先使用“关闭转写”或“豆包语音识别”。本地 Whisper 方案暂不做，因为安装和电脑适配成本较高。

## 三、豆包语音识别配置

1. 打开火山引擎豆包语音控制台。
2. 开通“录音文件识别 2.0”。
3. 进入“API Key 管理”，复制语音服务的 API Key。
4. 在 Obsidian 插件设置中选择“豆包语音识别”。
5. 填入 API Key。
6. 保持默认等待次数和等待间隔即可。
7. 在微信小程序录一条语音，回到 Obsidian 点击“立即同步”测试。

常见错误：

- `401`：API Key 类型不对，通常填成了火山方舟 Key。
- `403`：服务未开通、资源未授权、余额或免费额度未生效。
- `volc.bigasr.auc_turbo requested resource not granted`：插件版本旧了，需要更新到使用 `volc.seedasr.auc` 的版本。
- 转写为空：检查录音是否有声音，或等待任务处理完成后再次同步。

## 四、腾讯云 ASR 配置

1. 在腾讯云控制台开通“语音识别 ASR”。
2. 确认账号有可用免费额度或余额。
3. 创建 API 密钥，得到 `SecretId` 和 `SecretKey`。
4. 确认 CAM 权限允许调用 ASR 的录音文件识别接口。
5. 在 Obsidian 插件设置中选择“腾讯云 ASR 录音文件识别”。
6. 填入 `SecretId`、`SecretKey`。
7. 地域默认 `ap-shanghai`，普通话模型默认 `16k_zh`。
8. 点击“立即同步”测试语音转写。

常见错误：

- `SignatureFailure`：密钥不对或签名参数错误，先重新复制 SecretId/SecretKey。
- `UnauthorizedOperation`：CAM 权限不足或服务未开通。
- 没有转写结果：等待任务处理完成，或增加最大等待次数。

## 五、阿里云百炼 Qwen-Omni 配置

1. 打开阿里云百炼控制台。
2. 创建并复制 API Key。
3. 确认账号有可用免费额度或余额。
4. 在 Obsidian 插件设置中选择“阿里云百炼 Qwen-Omni”。
5. 填入 API Key。
6. 模型默认 `qwen3.5-omni-plus`。
7. 接口地址保持默认即可。
8. 点击“立即同步”测试语音转写。

常见错误：

- `401`：API Key 不对或复制缺失。
- `403`：服务未开通、账号权限或额度异常。
- 返回空文本：检查音频链接是否可访问、录音是否有声音。

## 六、插件更新说明

后续新版插件会统一放在本飞书文档中：

1. 打开本文档。
2. 找到“最新版插件下载”区域。
3. 下载 `wechat-obsidian-inbox` 插件文件夹或压缩包。
4. 覆盖 Obsidian 插件目录中的旧文件。
5. 关闭并重新开启 Obsidian 插件。
6. 再点击“立即同步”测试。

## 七、API Key 安全提醒

不要在群聊、公开文档、截图里暴露 API Key、SecretId、SecretKey 或飞书机器人 Webhook。  
如果已经发出来，建议立刻到对应平台删除旧 Key，重新生成一个新 Key。

## 八、反馈与建议

小程序首页提供“反馈与建议”按钮。MVP 阶段采用“微信客服实时沟通 + 云数据库留档”：用户提交后会先保存到云数据库 `feedback`，随后页面引导用户点击微信客服按钮，让开发者实时收到并回复。

如果反馈发送失败，云数据库仍会保存反馈记录，开发者可在云开发数据库中查看。
